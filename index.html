Colorful Image Colorization — Deep CNN for Automatic Image Colorization

A research project implementing a deep convolutional neural network to automatically colorize grayscale images. Inspired by the seminal paper Colorful Image Colorization (Zhang et al., ECCV 2016), the project explores multiple colorization strategies in CIE Lab color space and evaluates them both quantitatively and qualitatively.

Deep CNN CIE Lab K-means ImageNet PyTorch Computer Vision Research
View on GitHub
Color Space — CIE Lab: Uses K-means clustering on real color values from ImageNet to quantize the ab channels into 313 color bins, significantly reducing prediction complexity for the model.
Approach 1 — Regression: Baseline method treating colorization as a direct regression problem on CIE Lab channels.
Approach 2 — Classification-Centroid: Selects the highest-probability color bins and assigns their centroid ab values to corresponding pixels.
Approach 3 — Classification-Softmax: Uses top-k predicted color classes, computing the final ab value as a weighted sum with softmax probabilities as weights — producing more vibrant and spatially consistent colorizations.
Evaluation: Quantitative assessment via MSE & PSNR metrics; qualitative assessment via Visual Turing Test (fool rate between colorized vs. real images) and side-by-side visual comparisons across methods.

Feeding Frenzy — Big Fish Eats Small Fish

A Python remake of the classic Feeding Frenzy arcade game where you control a fish to eat smaller fish and grow bigger. Features a full progression system with leveling, boss encounters, power-ups, and a unique hand gesture control mode using computer vision — letting you steer your fish with real-time hand movements via webcam.

Python NumPy OpenCV (cv2) Hand Gesture Control Game Dev Pygame
View on GitHub
Core Gameplay: Control your fish to eat smaller fish and grow in size. The bigger you get, the more fish you can devour — classic predator progression loop.
Dash & Boost: Sprint mechanic lets you dash quickly to chase down prey or escape from larger predators, adding a layer of skill-based movement.
Level-up System: Collect experience stars dropped by eaten fish to level up, unlock new fish forms, and increase your size tier.
Boss Encounters: Face powerful boss fish at milestone levels — each with unique behaviors and patterns that challenge the player's reflexes.
Power-ups — Bombs & Stars: Pick up bombs to blast nearby fish and experience stars for burst XP gains, scattered across the ocean map.
Hand Gesture Control (cv2): Beyond WASD / arrow keys, the game supports real-time hand tracking via OpenCV — move your hand in front of the webcam to steer the fish, making gameplay feel entirely different and immersive.
